# Installation
Go to [willow-front-end repository](https://github.com/missionlane/willow-front-end) and then clone the repo using git.

Go inside the project:
```shell
cd willow-front-end
```
Ensure you have the correct node version:
```shell
node -v
# should print the version 14.16.1
```
Then install the dependencies with the following script
```shell
yarn install
```
Now you can start the project locally, using the next script:
```shell
yarn start-dev
```

# Pre-requirements
1. Have node v14.16.1
  - See how to install nvm [here](https://github.com/nvm-sh/nvm#installing-and-updating) and follow the instructions.
  - See how to install a `node` version using `nvm` [here](https://github.com/nvm-sh/nvm#installing-and-updating)
2. Have yarn in your shell
  - See how to install yarn [here](https://yarnpkg.com/getting-started/install)
